div { text-align : center ; } zorgt er voor dat de text gecentreerd is.

het heeft geen effect omdat "vertical-align: center" de "center" niet de juiste waarde is.

het past de verticale uitlijning aan met "top", "middle" en "bottom".

"vertically-centerd-text { hight: line-height:" is een simpel manier.

"vertical-center-content { display: table-cell;" is een veelgebuikte truc om meerdere regels tekst verticaal te centreren.

De klasse .tall is specifieker dan de .area.
