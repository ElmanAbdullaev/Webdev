Opdracht 1: De blokjes komen overeen met alle functies die je toevoegt in je code,
en de lijntjes verwijzen naar welke functies met welke code verbonden zijn.


Opdracht 2: Inspect element werkt inderdaad om dingen tijdelijk te veranderen totdat
je beslist om de pagina te refreshen.


Opdracht 3: Ze bestaan allemaal uit een body, header, footer en script. Als je de ene
accordeon open clickt wordt de code breder van de die dat je opent en komt de
volledige code tevoorschijn.Onderaan de pagina zien we de source code. Het
is langer want de code dat we in inspect element zien is alles.
